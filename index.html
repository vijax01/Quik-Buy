About Our E-Commerce Simulation

Project Overview

This project is a simulated e-commerce website built to demonstrate a complete online shopping experience using HTML, CSS, and JavaScript. It includes features like adding items to the cart, managing quantity, and storing data using the browser's localStorage.

Features

  • Add and remove products from the cart
  • Quantity and total price calculation
  • Data persistence using localStorage

Tech Stack

HTML • CSS • JavaScript • LocalStorage • FontAwesome • Google Fonts

Purpose

This project was created as a learning experience to understand front-end development, DOM manipulation, and how to manage data persistence using localStorage.
